# Codex Render Prompts

Purpose: dedicated codex art for entries that should not use briefing stills.

Style target for all renders:
- premium sci-fi codex illustration
- clean, high-detail, game-key-art quality
- cinematic lighting
- readable silhouette at mobile size
- polished, serious, not goofy
- no UI text, no watermark, no logo overlay
- not a briefing room screenshot
- not a pilot glamour portrait unless explicitly requested
- 3:4 or near-portrait composition preferred for codex panels

---
